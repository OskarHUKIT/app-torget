# Nytti Brand Assets

Place design assets here for the Nytti platform.

## Folder structure

- **logos/** – Logo files (logo.svg, logo.png, icon variants)
- **colors/** – Color swatches, palette reference

## What to add

- `logos/logo.svg` – Primary logo (vector)
- `logos/logo.png` – Logo for raster use
- `logos/icon-192.png`, `icon-512.png` – PWA icons (or keep in /public root)
- `colors/palette.json` – Hex codes, e.g. `{"primary": "#...", "secondary": "#..."}`
